# Description

Unofficial Dropbox Paper app for Mac built with [Electron](https://electronjs.org/).

<img src="https://raw.githubusercontent.com/dreamagicjp/paper-app/master/docs/paper.png">

# How to build

## Setup

```shell
npm install
```

## Run app

```shell
npm run start
```

## Build app

```shell
npm run package
```
